# <a name="recommended-adapters"></a>Adaptadores recomendados
> [!NOTE]
> Vamos a actualizar y realizar adiciones a estas recomendaciones de adaptadores Windows Mixed Reality pc. Vuelva a consultar periódicamente la versión más reciente.

El [Windows Mixed Reality equipo compatible](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md) puede no tener los puertos HDMI y USB 3.0 de tamaño completo necesarios para conectar el casco envolvente. Si ese es el caso, necesitará un adaptador. Algunos adaptadores también requieren la adición de un acoplador para conectarse al equipo o al cable del casco. Hemos enumerado los diferentes tipos de adaptadores disponibles y algunas recomendaciones sobre modelos específicos para empezar a trabajar.

## <a name="windows-mixed-reality-compatible-usb-bluetooth-adapters"></a>Windows Mixed Reality adaptadores de Bluetooth USB compatibles

Si el casco se lanzó después de 2018, no necesita un adaptador y un par de controladores para el casco.

Los dispositivos enviados en 2017 o versiones anteriores necesitan un adaptador para los controladores. Si el equipo no incluye Bluetooth para emparejar controladores de movimiento Windows Mixed Reality [,](controllers-in-wmr.md#pair-motion-controllers)puede usar un adaptador como el adaptador usb conectable Bluetooth microservicio de baja energía 4.0.

> [!NOTE]
> Microsoft no fabrica, prueba, certifica ni garantiza la funcionalidad, confiabilidad o calidad de ningún adaptador. Para cualquier pregunta, problema o problema con cualquier adaptador, debe ponerse en contacto con el vendedor o fabricante del adaptador.

## <a name="using-external-usb-30-hubs-with-windows-mixed-reality-headsets"></a>Uso de concentradores USB 3.0 externos Windows Mixed Reality cascos

Puede usar cascos Windows Mixed Reality con un concentrador USB 3.0 externo si el equipo no tiene suficientes puertos USB abiertos. El concentrador USB 3.0 externo debe tener alimentación externa a través de un adaptador de **CA** para proporcionar suficiente energía al casco.

Tenga en cuenta también las siguientes instrucciones para el uso de un concentrador USB 3.0 externo: 
* El concentrador USB debe ser un concentrador USB 3.0.
* El concentrador USB 3.0 debe estar conectado a un puerto USB 3.0 en el equipo.  
* El concentrador USB 3.0 debe tener alimentación externa (tener un adaptador de CA que proporciona alimentación adicional).
* Se sabe que el concentrador SuperSpeed Hub usb 3.0 conectable de 10 puertos con adaptador de alimentación de 48 W funciona bien con Windows Mixed Reality cascos y la mayoría de los equipos.
